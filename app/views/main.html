Welcome to another UMSL XTremeIT demo! We hope you enjoy exploring how web applications are built and have fun exploring the code on your own.

The Basics

Below are the core standards being used for this demo.

HTML5

HTML5 is the latest and greatest in HTML. Designed to be the content creator's language. It provides rich semantics and better compatability with old and new standards.

CSS3

CSS3 brings an increased ability to style and animate content without the need for JavaScript

JavaScript

JavaScript provides the foundation for creating robust interactive web pages and web applications, whether for a desktop or a mobile device.

The Toolset

Tools like the ones below help developers automate repetitive tasks and leverage the power of community works.

Node

Node is a JavaScript based server for all your server side scripting needs.

Grunt

Grunt is a JavaScript task runner. With it the developer can define various task and execute them repeatedly and consistently, removing some of the manual effort.

Yeoman

Yeoman is a scaffolding tool that helps you quickly build out whole sites or repetitive content with a little setup.

Bower

Bower helps keep track of external dependencies, like which version of jQuery, Angular, or Bootstrap is being used.

Karma

Spectacular Test Runner for JavaScript.

Angular

AngularJS is a toolset for building the framework most suited to your application development.

Bootstrap

Bootstrap is Twitter's responsive UI framework and comes with everything you need to create usable and nice looking web applications.

LESSCSS

LESS is a CSS-based markup that allows the developer to treat CSS in a more object oriented way. The markup allows the developer to do things that normal CSS can't, like create variables, or easily extend classes
